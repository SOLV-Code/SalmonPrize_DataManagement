# SalmonPrize_DataManagement

Repository for reorganizing and archiving the data sets



* DATA folder has a sub folder for each competition, and within that sub-folders for different systems (e.g., Fraser, Bristol Bay, Columbia) if the systems are handled separately in terms of data packages and competition timelines.

* For each data package, there are 2 folders: 
   - a folder with source files, notes, and any documentation that accumulates in response to questions
   - a folder with generated alternative versions of the data (e.g., reorganized by brood year)

* CODE folder has some scripts that implement the data reorg
